# 📘 Descripción general del proyecto

## 🚀 ¿Qué es OpenAPI Finder?

OpenAPI Finder es una aplicación web pensada para facilitar el descubrimiento de APIs públicas mediante una experiencia visual, ordenada y sencilla de usar.

El proyecto busca ser útil tanto para desarrolladores que necesitan APIs para sus proyectos como para juniors que quieren aprender contribuyendo a una aplicación real.

---

## ❓ Problema que resuelve

Muchas APIs públicas están dispersas, cuesta encontrarlas rápido y no siempre es fácil comparar su utilidad, categoría o características principales.

OpenAPI Finder busca resolver eso ofreciendo:

- una interfaz clara
- búsqueda rápida por nombre
- filtros por categoría
- tarjetas informativas
- una página de detalle por API

---

## 👥 Público objetivo

Este proyecto está orientado a:

- desarrolladores junior
- estudiantes de programación
- personas que buscan APIs para practicar
- usuarios que crean proyectos personales
- contribuidores que quieren iniciarse en open source

---

## 🧩 Funcionalidades principales

- listado de APIs públicas
- búsqueda por nombre
- filtro por categoría
- tarjetas con información resumida
- detalle ampliado de una API concreta

---

## 🎯 Alcance inicial

La primera versión del proyecto se centrará en:

- mostrar una lista inicial de APIs
- permitir búsqueda por nombre
- filtrar por categoría
- mostrar un estado vacío si no hay resultados
- preparar una base clara para futuras contribuciones
