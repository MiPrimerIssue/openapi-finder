# 🚀 OpenAPI Finder

**OpenAPI Finder** es una aplicación web open source diseñada para ayudar a desarrolladores a descubrir APIs públicas de forma rápida, clara y visual.

Este proyecto forma parte del ecosistema **MiPrimerIssue**, una iniciativa orientada a que desarrolladores junior hispanohablantes puedan contribuir a proyectos reales mediante tareas guiadas, progresivas y útiles para su aprendizaje.

---

## ✨ ¿Qué hace este proyecto?

OpenAPI Finder permite:

- 🔎 Buscar APIs públicas por nombre
- 🗂 Filtrar APIs por categoría
- 🧾 Mostrar información resumida de cada API
- 🌐 Consultar detalles ampliados de una API concreta
- 🧠 Aprender a contribuir en un proyecto open source real

---

## 🎯 Objetivo del proyecto

Encontrar APIs públicas útiles no siempre es fácil. Muchas están repartidas en diferentes sitios, sin una estructura clara o sin una forma cómoda de explorarlas.

El objetivo de OpenAPI Finder es ofrecer una experiencia simple y moderna para:

- descubrir APIs para proyectos personales
- practicar integración con servicios externos
- aprender a trabajar sobre un producto real
- facilitar primeras contribuciones open source a juniors

---

## 🧰 Tecnologías utilizadas

Este proyecto utiliza:

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **Supabase**
