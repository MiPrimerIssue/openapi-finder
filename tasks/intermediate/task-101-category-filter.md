# 🗂 Task 101 - Category Filter

## Nivel

Intermediate

---

## 🎯 Objetivo

Implementar un filtro por categoría para permitir al usuario ver solo las APIs que pertenecen a una categoría concreta.

---

## 🧠 Contexto

Además de la búsqueda por nombre, la aplicación necesita una forma rápida de limitar resultados por categoría y mejorar la navegación entre APIs.

---

## ✅ Requisitos

- Crear un selector o sistema de filtros por categoría
- Debe permitir:
  - mostrar todas las categorías
  - seleccionar una categoría concreta
- El filtro debe actualizar los resultados visibles
- Debe integrarse correctamente con el listado principal
- Debe mantener una experiencia simple y clara
