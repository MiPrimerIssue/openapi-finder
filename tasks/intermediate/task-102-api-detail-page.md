
# 🌐 Task 102 - API Detail Page

## Nivel

Intermediate

---

## 🎯 Objetivo

Crear una página de detalle para mostrar información ampliada de una API seleccionada.

---

## 🧠 Contexto

La lista principal mostrará solo información resumida. Para mejorar la experiencia del usuario, necesitamos una vista donde se pueda consultar más detalle de cada API.

---

## ✅ Requisitos

La página de detalle debe poder mostrar:

- nombre de la API
- descripción
- categoría
- soporte HTTPS
- información adicional disponible según la estructura de datos del proyecto

Además:

- debe tener una estructura clara
- debe ser fácil de leer
- debe integrarse con la navegación del proyecto
- debe mantener coherencia visual con el resto de la aplicación
