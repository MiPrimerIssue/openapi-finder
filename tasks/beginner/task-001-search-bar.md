# 🔎 Task 001 - Search Bar Component

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un componente reutilizable de búsqueda para filtrar APIs por nombre.

---

## 🧠 Contexto

La aplicación necesita una barra de búsqueda simple, clara y reutilizable para que el usuario pueda encontrar APIs rápidamente desde la página principal.

---

## ✅ Requisitos

- Crear un componente llamado `SearchBar`
- Debe incluir:
  - un campo de texto
  - placeholder
  - soporte para `value`
  - soporte para `onChange`
- Debe ser reutilizable
- Debe adaptarse bien a móvil y escritorio
- Debe seguir el estilo visual general del proyecto
