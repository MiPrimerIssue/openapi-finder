
# 📭 Task 003 - Empty State UI

## Nivel

Beginner

---

## 🎯 Objetivo

Crear un componente de estado vacío para mostrar cuando no haya APIs que coincidan con la búsqueda o los filtros.

---

## 🧠 Contexto

Cuando un usuario realiza una búsqueda y no encuentra resultados, la interfaz debe responder con un mensaje claro, amable y útil.

---

## ✅ Requisitos

El componente debe incluir:

- un mensaje claro
- una estructura visual limpia
- opcionalmente un icono o ilustración sencilla

Mensaje sugerido:

`No se han encontrado APIs. Prueba con otra búsqueda o categoría.`
