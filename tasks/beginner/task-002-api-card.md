
# 🧾 Task 002 - API Card Component

## Nivel

Beginner

---

## 🎯 Objetivo

Crear una tarjeta reutilizable para mostrar la información básica de una API pública.

---

## 🧠 Contexto

La pantalla principal mostrará varias APIs en formato tarjeta. Necesitamos un componente visualmente claro, reutilizable y fácil de mantener.

---

## ✅ Requisitos

La tarjeta debe mostrar:

- nombre de la API
- breve descripción
- categoría
- si soporta HTTPS

Además:

- debe ser responsive
- debe ser reutilizable
- debe tener una estructura visual clara
- debe encajar con una UI moderna y simple
