
# 📚 Tareas del proyecto

En esta carpeta se organizan las tareas disponibles para contribuir a **OpenAPI Finder**.

---

## 🪴 Niveles

- `beginner/` → tareas sencillas para empezar
- `intermediate/` → tareas con algo más de complejidad

---

## 🧩 Qué debe incluir una tarea

Cada tarea debe incluir:

- título
- nivel
- objetivo
- contexto
- requisitos
- archivos sugeridos
- criterios de aceptación

---

## ⚠️ Importante

No empieces una tarea sin haberla solicitado y haber sido asignado desde **MiPrimerIssue**.

---

## 🎯 Objetivo

Estas tareas están pensadas para que las contribuciones sean claras, acotadas y útiles tanto para el proyecto como para la persona que participa.
